LONG DESCRIPTION
The RMMSiteExtendedProperty enum defines the types of extended properties that can be retrieved for a site in the Datto RMM platform. These extended properties allow callers to request additional related data when fetching site information, such as the site's settings, variables, or filters.

## VALUES

The following values are defined for RMMSiteExtendedProperty:
