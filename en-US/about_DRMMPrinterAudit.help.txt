TOPIC
    about_DRMMPrinterAudit

SHORT DESCRIPTION
    Describes the DRMMPrinterAudit class and its methods for accessing printer audit data in Datto RMM.

LONG DESCRIPTION
    Datto RMM provides detailed audit snapshots for network printers. Use Get-RMMPrinterAudit to retrieve a DRMMPrinterAudit object for a printer device. For standard device audits, see DRMMDeviceAudit.
